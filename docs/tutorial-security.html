Tutorial: Setting Up Security

Setting Up Security

How it works

Security takes a user's permission and verify if all permissions stated by the command have been met. All security methods throw error if permissions are not met. Setting up security is optional.

Setting up bot check

This feature is used to check if bot has the set permissions. It makes sure that the bot has the permissions to execute the commands.

  1. Set botPermissions property to the desired permissions

    this.botPermissions = ["MANAGE_ROLES"];

  2. Create a try catch statement inside the execute method

    async execute(service){
  try {

  }catch(e){

  }
}

  3. Inside try statement call on botCheck

    await this.botCheck(service);

  4. Handle error, inside catch statement

    // It is suggested to send Error message to user
await service.sendEphemeral(e);
